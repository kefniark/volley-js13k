# Minimalist Volley

Currently in development ^^
Don't look too much at the code quality / readability for the moment

## Description
Try to make a small volley game in the 13KB limit (http://js13kgames.com/)


## Commands
Setup
```
npm install
```

Development (and open http://localhost:9000/):
```
npm run dev
```

Make a build
```
npm run build
```
